#### **Bedrock-Panel** 是一款基于 PHP 开发的网页管理面板，为了简化开服操作而进行的开服。

- 多用户 满足您多用户管理的需求
- 多实例 满足您多服务器管理的需求
- 扩展性 提供 API 接口，您可以随意扩展 ( WEB、PC、APP、QQbot )
- 隔离性 每个服务器之间我们都是隔离的，您无需担心数据冲突等问题
